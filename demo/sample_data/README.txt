QDARIA QRNG DEMO - SAMPLE DATA
===============================

This folder contains sample files for testing the demo application:

1. demo_document.txt
   - Sample confidential document
   - Use this to test Zipminator encryption
   - Demonstrates GDPR compliance features

2. test_image.txt (placeholder)
   - Can be replaced with actual image files
   - Demonstrates binary file encryption

3. financial_data.txt (placeholder)
   - Sample financial records
   - Shows secure data handling

Usage:
------
1. Launch the demo application
2. Navigate to the Zipminator tab
3. Drag and drop any file from this folder
4. Click "Encrypt File" to see the encryption demo
5. Download the encrypted file
6. Observe the security features in action

Note: This is a demonstration system. Do not use for production data.
