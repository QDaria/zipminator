## PRIMARY PROMPT

```
Create a modern, premium landing page for "Zipminator" - a quantum-secure encryption platform that combines real quantum hardware entropy with NIST-approved post-quantum cryptography (Kyber768).

DESIGN INSPIRATION:
- Primary reference: https://www.qbraid.com (quantum computing platform aesthetic)
- Style: Clean, futuristic, quantum-focused with dark mode
- Color palette: Deep space blacks with quantum purple/blue/pink gradients
- Typography: Bold, modern sans-serif (similar to Inter or Clash Display)
- Visual style: Premium SaaS/developer platform with quantum/tech aesthetic

HERO SECTION:
- Headline: "Quantum-Secure Encryption for the Post-Quantum Era"
- Subheadline: "Real Quantum Entropy + NIST FIPS 203 Cryptography"
- Visual: Animated quantum network/lattice background with glowing nodes and connections
- Two prominent CTAs: "Get Started Free" (primary) and "Request Demo" (secondary)
- Background: Dark gradient with floating quantum particles and interconnected lattice grid

QUANTUM PROVIDER SHOWCASE (inspired by qBraid's "All providers in one place"):
- Section title: "Connect to Real Quantum Hardware"
- Subtitle: "All quantum providers, unified API"
- Display 5 provider cards in a horizontal row:
  * IBM Quantum (127 qubits)
  * IonQ (11 qubits)
  * Rigetti (79 qubits)
  * AWS Braket
  * OQC (8 qubits)
- Each card should have: Provider logo placeholder, qubit count, and subtle glow effect
- Cards should have glassmorphism effect with subtle borders

TRUST SIGNALS SECTION:
- Title: "Trusted by Enterprises Securing Quantum-Safe Data"
- Display client logos in a 4x2 grid (use placeholder boxes):
  * Fortune 500 companies
  * Government agencies
  * Healthcare institutions
  * Financial services
- Include certification badges:
  * "NIST FIPS 203 Compliant" badge
  * "Memory-Safe Rust" badge
  * "127-Qubit Quantum Hardware" badge

KEY FEATURES (3-column grid):
- Feature 1: "Real Quantum Hardware"
  * Icon: Quantum chip/circuit
  * Text: "127-qubit IBM Brisbane with 15x cost reduction"
- Feature 2: "NIST Compliant"
  * Icon: Shield with checkmark
  * Text: "FIPS 203 approved Kyber768 encryption"
- Feature 3: "Memory Safe"
  * Icon: Lock/padlock
  * Text: "100% Rust eliminates entire vulnerability classes"

HOW IT WORKS (3-step visual process):
- Step 1: "Generate" → Quantum hardware illustration
- Step 2: "Encrypt" → Lattice/crystal structure
- Step 3: "Secure" → Shield/protection visual
- Connect steps with animated dotted lines or arrows

STATS BAR (4 metrics in a row):
- "0.034ms" - Encryption Speed
- "127 Qubits" - Quantum Hardware
- "NIST Level 3" - Security Standard
- "95%+ Coverage" - Test Quality

USE CASES (4 cards in 2x2 grid):
- Financial Services (banking icon)
- Healthcare (medical cross icon)
- Government/Defense (government building icon)
- Cloud Providers (cloud icon)

NAVIGATION HEADER:
- Left side: "QDaria" logo/wordmark (parent company)
- Center: Main navigation (Product, Pricing, Docs, Blog)
- Right side: "Sign In" and "Get Started" buttons
- Breadcrumb below: "QDaria > Products > Zipminator"

FOOTER:
- 5-column layout:
  * Column 1: QDaria logo + tagline
  * Column 2: Product links
  * Column 3: Resources links
  * Column 4: Company links
  * Column 5: Social media icons
- Bottom bar: Copyright, Privacy Policy, Terms, GDPR compliance

COLOR PALETTE:
- Primary: #6366f1 (quantum indigo/purple)
- Accent: #a855f7 (bright purple)
- Background: #0a0a0f (deep space black)
- Surface: #1a1a24 (dark card background)
- Text: #f3f4f6 (light gray)
- Gradients: Purple → Blue → Pink (use liberally)

DESIGN ELEMENTS:
- Glassmorphism cards (frosted glass effect)
- Subtle grid patterns in background
- Floating quantum particles (animated)
- Glowing borders on hover
- Smooth animations and transitions
- Quantum lattice/network visualizations
- Depth with subtle shadows and blur effects

TYPOGRAPHY:
- Headings: Bold, 64-72px for hero, gradient text effects
- Body: 16-18px, good readability
- Monospace for code/technical specs
- Generous line-height (1.6-1.8)

REFERENCE AESTHETIC:
Design should feel like a premium developer/quantum platform similar to:
- qBraid.com (primary inspiration)
- Vercel.com (clean, modern)
- Linear.app (sophisticated, minimal)
- Stripe.com (professional, trustworthy)
But with heavy quantum/cryptography theming (particles, lattices, quantum circuits)

TARGET AUDIENCE:
- Enterprise CTOs and security engineers
- Government/defense contractors
- Quantum computing enthusiasts
- Cryptography professionals

BRAND PERSONALITY:
- Cutting-edge and scientific
- Trustworthy and secure
- Premium and professional
- Futuristic but accessible

MOBILE RESPONSIVE:
- Stack sections vertically on mobile
- Collapse provider cards into carousel
- Hamburger menu for navigation
- Maintain visual hierarchy

Make it look premium, trustworthy, and quantum-forward. Think "the future of encryption is here."
```
