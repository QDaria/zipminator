## 2. IP Protection Costs

### 2.1 Trademarks (via Madrid Protocol — RECOMMENDED)

The **Madrid Protocol** lets you file one application through Patentstyret (Norway), designating EU + US in a single form. This is the cheapest multi-jurisdiction route.

| Item | ZIPMINATOR | QDARIA | Notes |
|------|-----------|--------|-------|
| **Norway base** (Patentstyret, 2 classes) | NOK 4,600 (~$430) | NOK 4,600 (~$430) | Class 9 (software) + Class 42 (SaaS) |
| **Madrid basic fee** (WIPO) | CHF 653 (~$730) | CHF 653 (~$730) | Standard character mark |
| **EU designation** (2 classes) | CHF 1,014 (~$1,130) | CHF 1,014 (~$1,130) | Covers 27 EU countries |
| **US designation** (2 classes) | CHF 714 (~$800) | CHF 714 (~$800) | Avoids US attorney requirement |
| **Subtotal per mark** | **~$3,090** | **~$3,090** | Norway + EU + US |
| | | | |
| **BOTH MARKS TOTAL** | | **~$6,180** | **<-- RECOMMENDED** |

**Timeline:** 8-18 months to full registration; protection starts from filing date.

**Alternative (direct filing, more expensive):**

| Route | Per mark (2 classes) | Both marks |
|-------|---------------------|------------|
| Patentstyret only | $430 | $860 |
| + EUIPO direct | +$1,000 | +$2,000 |
| + USPTO direct (requires US attorney) | +$2,700 | +$5,400 |
| **Total (direct)** | **$4,130** | **$8,260** |

Madrid Protocol saves ~$2,080 vs direct filing. **Use Madrid.**

### 2.2 Copyright Registration

| Item | Cost | Where | Coverage |
|------|------|-------|----------|
| Zipminator source code | $65 | US Copyright Office (eco.copyright.gov) | Worldwide + US statutory damages ($150K) |
| QDaria platform code | $65 | Same | Same |
| **Total** | **$130** | | |

> **Why US registration matters:** Copyright is automatic worldwide (Berne Convention). But US registration unlocks **$150,000 statutory damages per willful infringement** and attorney fee recovery. Without it, you can only claim actual damages (hard to prove). Given the CYBERELLUM situation, this is essential.

> **Norway:** Copyright is automatic under Åndsverkloven. No registration needed or available.
