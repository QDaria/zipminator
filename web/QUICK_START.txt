# ZIPMINATOR-PQC LANDING PAGE - QUICK START GUIDE

## Installation & Setup (5 minutes)

1. Navigate to directory:
   cd /Users/mos/dev/zipminator/landing-page

2. Install dependencies:
   npm install
   # or: bun install
   # or: pnpm install

3. Start development server:
   npm run dev
   # Server starts at: http://localhost:3000

4. Open browser:
   http://localhost:3000

## Deployment (Fast Track)

### Vercel (Recommended - 2 minutes):
1. Install: npm i -g vercel
2. Run: vercel
3. Follow prompts
4. Done! Your site is live

### Netlify:
1. Install: npm i -g netlify-cli
2. Run: netlify deploy --prod
3. Follow prompts

See DEPLOYMENT.md for detailed instructions.

## Customization

1. Colors: Edit tailwind.config.ts
2. Content: Edit component files in components/
3. Config: Edit lib/constants.ts
4. Branding: Update Navigation.tsx and Footer.tsx
